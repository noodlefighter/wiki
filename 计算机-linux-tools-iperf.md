iperf是用于测试网络通讯质量的工具



## 常用参数

* -s 服务端
* -c 客户端
* -i 显示报告的间隔
* -l 读写缓冲区
* -u 使用UDP而不是TCP
* -d 双向模式, 默认是客户端向服务端发数据
* -r 往复模式, 完成测试后服务端反向连接到-L指定的端口



## 简单用法

测试10秒, 每秒打印一次报告.

服务端

```
iperf -s
```

客户端

```
ipref -c [服务端地址] -t 10 -i 1
```

默认走5001端口
