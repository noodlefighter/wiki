## Rust工具

### Rustup

Rust工具链管理器，方便安装、更新、切换工具链。

```
# 安装stable/beta/nightly版本
rustup install stable
# 切换默认工具链为相应版本
rustup default stable
```

### cargo

Rust的包管理器。

```
# 新建包
cargo init
# 构建包
cargo build
```
