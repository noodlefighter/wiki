

## 燕麦南瓜饼

材料：

* 南瓜300g
* 燕麦80g-100g（根据南瓜水分而定）
* 葡萄干（曼越莓）

步骤

1. 南瓜蒸熟捣烂，加燕麦变得足够稠；

2. 烤盘上一层油；

3. 搓球，裹一层燕麦再放到烤盘里，压扁；

4. 180度烤20分钟



## 煎猪扒

```
里脊肉300g 分成三份 拍松
生粉2勺
糖2勺
酱油2勺
水4勺
油2勺

腌一晚，煎至金黄
```



## 猪肉饼

适合冷冻常备

```
肉200g（瘦肥比7:3）
糖5g
盐1.5g
蛋清15g
玉米淀粉10g
蚝油、鲜菇粉
```



## 面饼

```
中粉100g
蛋x1
糖10g
水110g
葱花

混合后醒15min即可煎，分多次煎，不然会厚，也容易焦
```
