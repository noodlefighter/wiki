## 芒果西米椰奶糕

```
基底：椰浆200ml，牛奶200ml，糖20g，吉利丁片20g
其他：西米30g，芒果500g，椰蓉少量撒上层
第1层：夹西米
第2层：夹芒果
```



## 椰奶西米露

```
西米150g
椰浆400ml
奶800ml
水200ml
糖50g
```



## 薏米水

```
（两人份）
薏米50g
水700g
糖8g

高压锅20分钟，闷20分钟
```



## 红糖姜汁奶茶

```
淡奶 25ml
水 100ml
奶 100ml
红茶 3g
姜 15g 切片

水里放姜->水开->关火->放红茶闷3分钟->加奶、淡奶
```



## 热巧

TIP: 可以直接放到水杯里，水浴加热不用洗锅

```
奶200g
水100g
黑巧(85%法芙娜) 40g
```



## 绿豆冰棒

```
[绿豆水]
绿豆:90g + 水:280g | 高压锅煮45min

[绿豆牛奶浓汤]
绿豆水 + 奶:100g + 糖:30g | 混合 | 锅开盖煮干

[绿豆冰棒]
淡奶油80g | 低温打发 + 绿豆牛奶浓汤 | 混合 | 装到模具中 | 冰箱冷冻
```
